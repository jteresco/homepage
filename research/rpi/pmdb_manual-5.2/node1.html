This version of the PMDB Manual is no longer maintained! Please see http://www.cs.williams.edu/~terescoj/research/pmdb_manual for a more recent edition.

PMDB 5.2 works with mesh/4.2 and model/1.2. That version of the mesh database has no pNode objects or node operators. This version of PMDB is compatible with those. It also is the first PMDB version which uses the new data attachment implementation and can take advantage of the mesh modification callbacks.
