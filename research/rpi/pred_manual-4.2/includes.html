Include Files

To use pred, you need to include the following file: 
pred.h
(located in /dept/scorec/software/pred/4.2/include).

If using the SCOREC common makefile format, a typical INCLUDES variable for use with pred is as follows: 

DEPS =  \
        model/mesh/4.2 \
        model/model/1.2 \
        util/util/0.4 \
        parallel/pmdb/5.2 \
        parallel/comm/2.2 \
        meshing/meshTools/3.1 \
        meshing/refdref/3.3 \
	parallel/pred/4.2
