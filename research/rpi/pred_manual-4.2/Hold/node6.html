Modifications Since Last Documented Version

  • Specialized instances of the library have been eliminated. This decouples pred from the balancers it uses. Applications only need to link in balancers they will actually run predictively.
  • The following operators have been depreciated:
    • pred_set_ref_balancing - predictive balancing is now only set using pred_set_ref_bal_func.
    • pred_check_env - predictive balancing can no longer be set from an environment variable.
    • pred_set_parmetis_params - this function was tied to a particular balancer.
