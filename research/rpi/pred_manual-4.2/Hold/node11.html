Uniform Element Workload Predictive Balancing

The following example program successfully builds and runs with the provided makefile (6-22-99):

  • Files to build the program (tar file: example.tar) may be found in
    /home/35/ziantl/develop/parallel/pred/4.2/pred/src/example on the SP.

Untarring the file will create a directory named example. The Makefile included in the tar file uses the SCOREC common makefile and assumes that library paths are set up accordingly. To build the sample executable, type 

gmake BUILDSYS=sp2gcc PARALLEL=mpi NODEP=1 MODELER=parasolid
This will create the appropriate bin/$(ARCHOS) subdirectory in the example directory automatically before generating the executable. The program may be run on the SP by typing: 
poerun -procs 2  bin/ibm_4/pred.test_parasolid.exe 
shocktube single <number>
where <number> is one of the following:

0 do not perform predictive balancing
1 perform predictive ITB balancing
2 perform predictive k-way partitioning using ParMetis [1]
