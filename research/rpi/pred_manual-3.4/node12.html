Using pred with ParMetis

  In order to use pred with ParMetis, the user must do the following:
  • add -DPARMETIS to the compile line
  • add the following to the list of includes:
    • -I$(SCSOFT_HOME)/scorec2metis/2.0/include 
-I$(SCSOFT_HOME)/ParMetis/1.0/include
  • add the following to the list of libraries to link in:
    • -L$(SCSOFT_HOME)/scorec2metis/2.0/lib/ibm6000mpi -lscorec2metis-O
-L$(SCSOFT_HOME)/ParMetis/1.0/lib/ibm6000mpi -lparmetis
-L$(SCSOFT_HOME)/MeTiS/2.0/lib/ibm6000 -lmetis
  • link with one of the pred_parmet* instances of pred
