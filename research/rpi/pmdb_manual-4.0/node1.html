Note for PMDB 3.2 users: When linking with PMDB 4.0, all other source code which include pmshops.h should be compiled using the PMDB 4.0 version of pmshops.h. It should work even if you do not recompile, but you may end up calling bounce routines which bind to the old operator names and call the new. If you recompile, macro replacement will allow the new routines to be called directly without any source code changes.
