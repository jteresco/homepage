SCOREC Mesh Structures to ParMetis Structures Library (scorec2metis)

The library contains routines that convert a mesh stored in SCOREC mesh format into structures usable by the ParMetis partitioning library developed at the University of Minnesota [2].

This documentation covers version 2.2 of the library. It has been tested on the RPI SP and Sun workstations. The library is located in
/dept/scorec/software/parallel/scorec2metis/2.2 on the SP. Contact ziantzl@cs.rpi.edu if you wish to run scorec2metis on another platform.
